# Contributing

Keep the website small, direct and accurate. It should explain IR Portugal and link to the community space, not grow into a project-management platform or pretend the group is an institute.

Before opening a pull request, run:

```bash
npm run format:check
npm run check
npm run lint
npm run build
npm run check:links
```

Do not add private data, fabricated activity, placeholder organisations, analytics or unnecessary dependencies.
